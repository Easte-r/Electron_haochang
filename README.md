# Electron_netEase_cloud
模仿网易云音乐 <br />
:blush:
# dependent
react + react-router + webpack + electron<br />
注意: <br />
node 版本至少 6.11.1 <br />
# 安装依赖
npm i
# 浏览器中预览（nodejs开启的本地服务器）
node http.js <br />
浏览器访问localhost:5000
# PC 端预览
npm run preview
# 构建应用
npm run packager <br />
会在当前目录创建win64位的应用
# tips:
electron 可能需要重新安装
